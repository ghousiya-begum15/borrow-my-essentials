# NEED NOW – Hyperlocal Borrowing Platform (Frontend Prototype)

## Overview
NEED NOW is a mobile-first frontend prototype developed during a hackathon. The platform is designed to connect users with nearby everyday essentials such as chargers, books, and headphones through a hyperlocal sharing model.

This project focuses on user interface design and user flow simulation for a borrowing-based ecosystem.

## Problem Statement
Many individuals experience urgent need for small essential items while away from home. Traditional solutions such as purchasing replacements or borrowing from strangers are inconvenient and costly. NEED NOW aims to provide a structured and community-driven borrowing alternative.


## Features (Frontend Prototype)
- Splash screen and login interface
- Home dashboard with item browsing
- Search functionality (UI simulation)
- Borrow request flow screens
- Profile and payment UI screens
- Map interface design (visual simulation)
- Chat and notification interface mockups
- Borrower and lender interaction screens

Note: Real-time data integration, payment processing, and backend services were conceptualised but not implemented in this prototype.
